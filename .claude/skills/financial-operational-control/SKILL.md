# Skill: financial-operational-control

## Cuándo usar este skill
Úsalo cuando el usuario necesite cuantificar el impacto económico de problemas u oportunidades operativas en inventario, compras, ingresos o eficiencia.

## Objetivo
Separar con claridad lo ya materializado de lo todavía potencial, para que Dirección pueda decidir con criterio financiero y sin mezclar conceptos.

## Taxonomía financiera obligatoria
Cada cifra debe clasificarse en una sola categoría principal:
- **Ahorro realizado:** gasto que ya se redujo y puede comprobarse.
- **Ahorro potencial:** oportunidad identificada pero no capturada.
- **Costo evitado:** desembolso futuro que probablemente no ocurrirá si se actúa.
- **Ingreso recuperable:** cargo o ingreso que puede rescatarse.
- **Capital liberable:** dinero inmovilizado que puede convertirse en liquidez.
- **Riesgo económico expuesto:** dinero en peligro si no se corrige el problema.

## Reglas obligatorias
- No mezclar ahorro con ingreso recuperable.
- No llamar ahorro a una compra evitada si la acción aún no ocurrió.
- Declarar período, fuente y estado de captura del valor.
- Si el dato no es suficiente, entregar fórmula y supuestos visibles.
- Diferenciar impacto puntual, mensual y anualizado.

## Marcos de análisis

### 1. Capital inmovilizado
Aplicar a inventario con DOI alto, baja rotación o sobrecompra.

**Lectura esperada:**
- valor inmovilizado actual
- porción potencialmente liberable
- riesgo de deterioro o vencimiento

### 2. Compra evitada
Aplicar a balanceo inter-compañía, uso de excedentes o cancelación de OCs innecesarias.

**Lectura esperada:**
- monto de compra evitada
- plazo de captura
- restricciones operativas para materializarla

### 3. Fuga de ingresos
Aplicar a diferencias entre consumo real y cuenta facturada.

**Lectura esperada:**
- ingreso recuperable inmediato
- pérdida histórica ya materializada
- causas raíz y área responsable

### 4. Merma y deterioro
Aplicar a vencimientos, obsolescencia o descuentos tácticos para evitar pérdida total.

**Lectura esperada:**
- valor perdido
- valor salvable
- costo de no actuar

### 5. Eficiencia operativa con impacto económico
Aplicar a reducción de reproceso, urgencias, tiempos muertos o dependencia manual.

**Lectura esperada:**
- horas o ciclos reducidos
- costo aproximado asociado
- capacidad liberada para actividades de mayor valor

## Formato de salida
1. Resumen ejecutivo financiero
2. Lectura del valor por categoría
3. Qué dinero ya se perdió, cuál se puede recuperar y cuál puede evitarse
4. Riesgos económicos y de control
5. Recomendación priorizada
6. Tabla de trazabilidad financiera

## Qué evitar
- sumatorias que mezclen categorías incompatibles
- presentar escenarios especulativos como dato confirmado
- conclusiones financieras sin sustento de fuente o fórmula
