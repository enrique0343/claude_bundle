# Skill: process-documentation-hospital

## Cuándo usar este skill
Úsalo cuando el usuario necesite formalizar un proceso, una decisión, un control o una práctica operativa en un documento claro, ejecutable y auditable.

## Objetivo
Convertir conocimiento tácito o análisis sueltos en documentación operativa que sirva para ejecutar, capacitar, controlar y auditar.

## Tipos de entregable
- SOP operativo
- instructivo de usuario
- minuta de comité
- política o lineamiento
- matriz RACI
- flujo técnico de proceso
- evidencia de control

## Estructura mínima obligatoria para procesos
- objetivo
- alcance
- actores
- disparador
- entradas
- proceso
- salidas
- riesgos
- controles

## Reglas obligatorias
- No inventes pasos operativos que no estén confirmados.
- Si falta información, deja supuesto visible o sección pendiente.
- Separa con claridad hecho, decisión, recomendación y control.
- Redacta para uso real de la operación, no como ensayo.
- Si el proceso toca inventario, farmacia, trazabilidad o paciente, marca implicación JCI cuando corresponda.

## Marcos de documentación

### 1. SOP
Usar cuando se necesite estandarizar ejecución repetitiva.

**Debe dejar claro:**
- quién ejecuta
- cuándo se ejecuta
- evidencia generada
- punto de control

### 2. Instructivo
Usar cuando se necesite explicar cómo realizar una tarea puntual en sistema o proceso.

### 3. Minuta
Usar cuando se necesite registrar decisión, responsables y fechas compromiso.

### 4. Política o lineamiento
Usar cuando se necesite fijar criterio rector, excepción y responsable de cumplimiento.

### 5. RACI
Usar cuando exista ambigüedad entre áreas o roles.

## Criterios de buena documentación
- legible en una sola pasada
- ejecutable sin interpretación excesiva
- auditable con evidencia identificable
- útil para entrenamiento de personal nuevo

## Formato de salida
1. Tipo de documento
2. Documento estructurado
3. Riesgos y controles
4. Vacíos de información pendientes

## Qué evitar
- redacción académica
- pasos ambiguos
- controles sin evidencia observable
